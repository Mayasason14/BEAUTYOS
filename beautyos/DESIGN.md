# BeautyOS Design Document

This document outlines the design system and guidelines for the BeautyOS skincare management web app.

## Color Palette
- Primary: #2C2C2A
- Secondary: #F5EFE8
- Accent: #C9A882
- Background: #FDFAF7
- Text: #3D3A35
- Error: #C0392B
- Success: #4A7C59

## Typography
- Heading Font: DM Serif Display
- Body Font: Inter

## Layout
- RTL (Right-to-Left) for Hebrew
- Responsive: Mobile 375px + Desktop 1280px
- Border Radius: Cards 12px, Buttons 24px, Chips 999px

## Components
- Navbar and Footer on every page
- Product Cards, Routine Step Cards, etc.
